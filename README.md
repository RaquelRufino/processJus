# Crawler ESAJ!

#### Tutorial
```
    $ git clone https://github.com/RaquelRufino/processJus.git
    $ cd processJus
    $ pip install -r requirements.txt
    $ cd service
    $ python3 server.py
```
#### API
```
    locahost:8000
    POST:
        {
    	"number": "1002298-86.2015.8.26.0271",
    	"state" : "sp"
        }
```
